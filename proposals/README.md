# 💰 הצעת תמחור — הבונקר

## 📋 סיכום ההצעה

3 חבילות שירות לאולפן הקלטת פודקאסט "הבונקר", על בסיס מחקר שוק (Israel · 2025-2026).

### 📦 חבילה 1 — רק צילום
- **₪1,200 / שעה** (עד 3 משתתפים)
- אולפן + 2 מצלמות 4K + טכנאי + multi-track audio
- מסירת קבצים גולמיים תוך 24 שעות
- **קהל יעד:** פודקאסטרים שעורכים בעצמם

### 📦 חבילה 2 — צילום + עריכה ⭐ (המומלצת)
- **₪2,400 / פרק** (90 דקות הקלטה + עריכה מלאה)
- כל מה שבחבילה 1 + **גרסה ראשונית באותו יום**
- 5 רילסים + פתיח/סגיר + כתוביות
- פורטל לקוח עצמי לסימון רילסים + 3 סבבי תיקונים
- מסירה סופית תוך 5 ימים
- **קהל יעד:** פודקאסטרים מקצועיים, יועצים, מנכ"לים

### 📦 חבילה 3 — ריטיינר Full Service
- **₪7,500 / חודש** (4 פרקים = ₪1,875/פרק, חסכון 21%)
- כל מה שבחבילה 2 × 4 פרקים
- **העלאה אוטומטית** ל-Spotify, Apple, YouTube + Instagram + TikTok
- ספרייה פרטית + אנליטיקס חודשי + תעדוף עריכה
- WhatsApp ישיר לצוות
- **קהל יעד:** פודקאסטרים פעילים שרוצים להתמקד בתוכן

## 🎯 USPs שמצדיקים את התמחור

1. **באותו יום** — אף מתחרה בישראל לא מציע
2. **פורטל לקוח** — הלקוח מסמן רילסים בעצמו דרך אפליקציה
3. **ניתוב אוטומטי** — קבצים זורמים אוטומטית בין צוות הצילום, העריכה והלקוח
4. **איכות מקצועית** — 2 מצלמות 4K, מיקרופונים אולפניים

## 📊 הצדקה תמחורית

על בסיס מחקר מתחרים:
- **Liron Music** "3 מצלמות + עריכה + 2 רילסים": ₪1,500/שעה → ₪2,250/90ד׳
- **VideoLab MasterPod**: ₪2,500 (3 רילסים, 3-5 ימי מסירה)
- **שוק אמריקאי** (לרפרנס): $1,250-5,000/חודש = ₪4,500-18,000

הבונקר ב-₪2,400/פרק נמצא במחיר תחרותי **+₪150 בלבד** מ-Liron, אבל נותן **+3 רילסים, מסירה מהירה ב-7 ימים, כתוביות, ופורטל לקוח** — שווי שוק של ~₪900.

## 🛠️ התאמות במערכת בעקבות התמחור

כדי לתמוך במודל המלא, נצטרך להוסיף:

1. **שדה `package` במודל לקוח** — כבר קיים! (photo_only / photo_edit / retainer)
2. **שדה `included_reels` במודל לקוח** — חדש (default: 0 לchat photo_only, 5 לphoto_edit, 20 לretainer)
3. **שדה `episodes_per_month` בריטיינר** — חדש (default: 4)
4. **תוספות שנרכשו לפרק** — טבלת `episode_addons` חדשה
5. **חישוב חיוב חודשי** — view/function ב-DB
6. **דף תמחור פנימי באדמין** — שייצר הצעה אוטומטית ללקוח חדש
