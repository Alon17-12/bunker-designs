## 🏗️ Tech Stack

- **Frontend:** Next.js 15 + TypeScript + Tailwind (RTL native)
- **Backend:** Supabase (Postgres + Auth + RLS)
- **Video:** Bunny Stream (hosting + CDN + signed URLs)
- **Automation:** Make.com + Green API (WhatsApp) + Gmail
- **Hosting:** Vercel
- **Kanban:** Monday.com

## 📅 Roadmap

| Phase | Duration | Focus |
|---|---|---|
| **MVP** | 6 weeks | Foundation, Admin, Video, Client Portal, Automation, Launch |
| **Phase 2** | 4-8 weeks | AI (SRT, Highlights, Trailer), Retainer Portal, Editor Role |
| **Phase 3** | 8-12 weeks | Booking system, Auto-distribution (YouTube/Spotify/IG/TikTok) |
